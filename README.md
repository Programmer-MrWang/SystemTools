# SystemTools

> 提供适用于Windows的一些行动和其他实用工具。

> [!NOTE]
> - 由于开发者手头并没有更多的不同Windows版本供测试，部分功能在某些Windows版本可能上无法正常工作。如遇此情形请提交 issue 反馈，并注明日志和诊断信息。
> - 已知输入UI存在丢失记录问题，目前版本请确保全ClassIsland每种行动仅使用了一个。本人对UI不熟且无足够精力，欢迎提出PR修复此问题。
 
## 目前实现的功能

### 行动：
 
**模拟操作**
 - 按下 Alt+F4
 - 按下 Alt+Tab
 - 按下 Enter 键
 - 按下 Esc 键
 - 按下 F11 键
 - 键入内容
 - 模拟点击

**显示设置**
 - 复制屏幕
 - 扩展屏慕
 - 仅电脑屏幕
 - 仅第二屏墓
 - 黑屏html

**电源选项**
 - 计时关机
 - 取消关机计划
 - 锁定屏幕

**系统个性化**
 - 切换壁纸
 - 切换主题色

**实用工具**
 - 全屏时钟
